Guia de Cores

Formatos de Cor

  • HEX: #RRGGBB - Web standard
  • RGB: rgb(r, g, b) - Vermelho, Verde, Azul
  • HSL: hsl(h, s%, l%) - Matiz, Saturação, Luminosidade
  • HSV: hsv(h, s%, v%) - Matiz, Saturação, Valor
  • CMYK: cmyk(c, m, y, k) - Impressão

Acessibilidade

  • WCAG AA: Razão de contraste mínima 4.5:1
  • WCAG AAA: Razão de contraste mínima 7:1
  • Texto grande: Mínimo 3:1 para AA
  • Componentes UI: Mínimo 3:1 para AA

Teoria de Cores

  • Complementar: Cores opostas no círculo cromático
  • Triádica: Três cores equidistantes
  • Análoga: Cores vizinhas no círculo
  • Monocromática: Variações da mesma cor

Perguntas Frequentes

Qual a diferença entre HSL e HSV?

HSL usa Luminosidade (quanta luz/preto), enquanto HSV usa Valor (quanta cor/branco). HSL é mais intuitivo para design.

Por usar HEX em vez de RGB?

HEX é mais compacto e amplamente suportado. RGB é mais legível para humanos e permite ajustes individuais.

O que é razão de contraste?

É a diferença de luminosidade entre duas cores, essencial para acessibilidade e legibilidade.

Como escolher cores acessíveis?

Use o verificador de contraste e garanta WCAG AA mínimo. Teste com diferentes tipos de daltonismo.
