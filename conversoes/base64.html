Guia Base64

O que é Base64?

Base64 é um método de codificação que converte dados binários em texto ASCII, usado para transmitir dados em canais de texto.

Quando usar Base64?

  • Embedder imagens em HTML/CSS
  • Transmitir dados binários em JSON/XML
  • Armazenar arquivos em bancos de dados
  • Email attachments
  • Data URLs em desenvolvimento web

Limitações

  • Aumenta o tamanho em ~33%
  • Não é criptografia (apenas codificação)
  • Performance para arquivos grandes
  • Compatibilidade com browsers antigos

Exemplos de Uso

HTML Image Embed

<img src="data:image/png;base64,iVBORw0KGgoAAAANS...">

CSS Background

background: url(data:image/png;base64,iVBORw0KGgoAAAANS...);

JavaScript Data URL

const img = 'data:image/png;base64,iVBORw0KGgoAAAANS...';

Perguntas Frequentes

Base64 é seguro para dados sensíveis?

Não! Base64 é codificação, não criptografia. Qualquer pessoa pode decodificar Base64 facilmente.

Qual o tamanho máximo de arquivo?

Limitamos a 10MB para performance. Arquivos maiores podem causar lentidão no navegador.

Posso usar Base64 para imagens em produção?

Sim, mas apenas para imagens pequenas. Imagens grandes devem ser servidas como arquivos separados.

Como identificar Base64?

Base64 geralmente termina com = ou == e contém apenas caracteres A-Z, a-z, 0-9, +, /.
